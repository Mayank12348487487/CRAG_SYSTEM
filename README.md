# 🧠 CRAG System (Corrective Retrieval Augmented Generation)

## 🚀 Features
- RAG + CRAG pipeline
- Long-term & short-term memory
- MongoDB integration
- LangChain + LangGraph
- Multi-document support
- AI Agent workflows

## 🏗️ Tech Stack
- Backend: FastAPI / Python
- Frontend: React / Vite
- Database: MongoDB
- LLM: MetaLLAMA 
## ⚙️ Setup

### 1. Clone Repo
git clone https://github.com/your-username/crag-project.git
cd crag-project

### 2. Backend
cd backend
pip install -r requirements.txt
cp .env.example .env
python main.py

### 3. Frontend
cd frontend
npm install
cp .env.example .env
npm run dev

## 🧠 Architecture
- Retriever → Validator → Corrector → Generator
- Memory layer (short + long)
